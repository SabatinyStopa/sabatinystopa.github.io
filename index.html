About Me

I’m a Gameplay Software Engineer specializing in creating immersive and dynamic game systems using Unreal Engine (C++) and Unity (C#). I love transforming creative concepts into interactive mechanics and experiences that engage players.

Throughout these projects, I’ve contributed to gameplay programming, system architecture, and user experience design across both Unreal Engine, Unity using c# or just pure c++. My work consistently focuses on delivering smooth, responsive gameplay and scalable systems with clean code.

  • Developed AI navigation, camera systems, and dynamic pathfinding behaviors.
